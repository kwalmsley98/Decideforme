# Decide For Me App

React + Vite app with:
- Home category screen
- Mood selection screen
- Claude-powered AI answer screen
- Supabase auth (login/signup) and decision history
- Stripe subscription checkout (Plus and Pro, both with 7-day trial)

## 1) Install prerequisites

- Node.js 20+
- npm

## 2) Setup env

1. Copy `.env.example` to `.env`
2. Fill in:
   - `VITE_SUPABASE_URL`, `VITE_SUPABASE_ANON_KEY`
   - `VITE_API_BASE_URL` (`http://localhost:8787` for local dev)
   - `ANTHROPIC_API_KEY` (server-side secret)
   - `STRIPE_SECRET_KEY` (server-side secret)
   - Stripe price IDs (`VITE_STRIPE_PRICE_PLUS_ID`, `VITE_STRIPE_PRICE_PRO_ID`)
   - `APP_BASE_URL` (frontend URL, e.g. `https://decideforme.org`)
   - `CORS_ALLOWED_ORIGINS` (comma-separated allowlist)

## 3) Setup Supabase table

Run `supabase.sql` in your Supabase SQL editor.

## 4) Run

```bash
npm install
npm run dev
```

Frontend: `http://localhost:5173`  
API: `http://localhost:8787`

## Notes

- Stripe checkout session is created by `POST /api/create-checkout-session`.
- Claude decision is generated by `POST /api/decide`.
- `POST /api/stripe-webhook` is stubbed and ready for event handling if you want to persist subscription status in Supabase.
- Frontend now reads API target from `VITE_API_BASE_URL`; in production this should be your hosted backend URL.
- Frontend must never receive `ANTHROPIC_API_KEY` or `STRIPE_SECRET_KEY`. Keep those only on your backend host.
- API health checks:
  - `GET /api/health` (summary)
  - `GET /api/health/live`
  - `GET /api/health/ready`
- Server startup now fails fast if required env vars are missing or invalid.

## Deploy outline

- **Netlify (frontend)**:
  - Build: `npm run build`
  - Publish dir: `dist`
  - Add only frontend-safe env vars in Netlify UI (`VITE_*` public values)
  - Do not set `ANTHROPIC_API_KEY`, `CLAUDE_API_KEY`, or `STRIPE_SECRET_KEY` on the Netlify frontend site
- **Backend host (Render/Railway/Fly)**:
  - Run `node server/index.js`
  - Add server env vars (`ANTHROPIC_API_KEY`, `STRIPE_SECRET_KEY`, `APP_BASE_URL`, `CORS_ALLOWED_ORIGINS`)
- Set `VITE_API_BASE_URL` in Netlify to your backend URL.
